Contenu du cours

Le cours se focalise sur l'analyse de fonctions réelles à une seule variable ainsi que sur les suites numériques réelles.
Concernant le suites, il s'agit de revoir les fondamentaux : monotonies, limites, suites récurrentes et des théorèmes de convergence sur les suites ainsi que des équivalents classiques.
Pour ce qui est des fonctions, il s'agit de reprendre l'études des fonctions classiques, circulaire ou encore trigonométriques : limites, continuité, dérivation, intégration sur un segment, et une introduction aux développements limités de fonctions usuelles. Quelques grands théorèmes classiques seront présentés et parfois démontrés dans le cadre de ce cours.

Une fiche de TD complète avec les corrections est disponible pour vous permettre de vous entraîner. Une séance de TD sera également consacrée à une partie mise en pratique sous Python afin d'illustrer les résultats vus en cours.
